# blog-gatsby-template

Gatsby で作成したブログテンプレートです。
[デモ](https://seelx3.github.io/blog-gatsby-template/)

## 特徴

- Gatsby v5
- TypeScript
- Tailwind CSS

## 開発手順

```
asdf install // nodejs, yarn のバージョン指定

yarn // パッケージインストール
yarn develop // 開発サーバー起動
yarn deploy // デプロイ
```
